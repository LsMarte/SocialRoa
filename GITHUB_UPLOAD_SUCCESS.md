## 📋 **Para Usar el Repositorio**

### 1. **Clonar**
```bash
git clone https://github.com/LsMarte/SocialRoa.git
cd SocialRoa
```

### 2. **Instalar Backend**
```bash
cd SocialRoa/backend
npm install
npm start
```

### 3. **Instalar Frontend**
```bash
cd SocialRoa/frontend-new
npm install
npm start
```

### 4. **Acceder**
- Frontend: `http://localhost:3000`
- Backend API: `http://localhost:3001`
